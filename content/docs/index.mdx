## What is Arch Network?

Arch Network is a computation environment that enhances Bitcoin's capabilities by enabling complex operations on Bitcoin UTXOs through its specialized virtual machine. Unlike Layer 2 solutions, Arch Network provides a native computation layer that works directly with Bitcoin's security model.

## Prerequisites

Before you begin, ensure you have:

- Node.js v19+ ([installation guide](/docs/quick-start/requirements))
- Rust (latest stable)
- Docker for local development
- Basic understanding of [Bitcoin UTXOs](/docs/program/utxo)

## Core Architecture

### How Arch Works

Arch Network consists of three main components:

1. **Network Layer**
   - [Network Architecture](/docs/setup-infrastructure/network-architecture)
     - [Bootnode](/docs/setup-infrastructure/network-architecture): Network discovery and peer management
     - [Leader Node](/docs/setup-infrastructure/network-architecture): Transaction coordination
     - [Validator Nodes](/docs/setup-infrastructure/network-architecture): Program execution

2. **Bitcoin Integration**
   - [UTXO Management](/docs/core-concepts/bitcoin-integration)
     - Transaction tracking
     - State anchoring
     - Ownership validation
   - [RPC Integration](/docs/core-concepts/bitcoin-integration)
     - Bitcoin node communication
     - Transaction submission
     - Network synchronization

3. **Computation Layer**
   - [Programs](/docs/program/program)
     - [Instructions](/docs/program/instructions-and-messages)
     - [Accounts](/docs/program/accounts)
     - [System Calls](/docs/program/syscall)
   - [Transaction Processing](/docs/sdk/processed-transaction)
     - Message validation
     - State updates
     - UTXO management
