# RPC Reference

This section documents Arch's HTTP RPC methods. Methods are grouped by category to make it easy to find what you need.

## Categories

- Node Status
  - get-version
  - is-node-ready
  - get-peers
  - get-best-block-hash
  - get-best-finalized-block-hash
  - get-block-count
- Blocks
  - get-block-hash
  - get-block
- Accounts
  - get-account-address
  - get-all-accounts
  - read-account-info
  - get-program-accounts
- Transactions
  - send-transaction
  - send-transactions
  - get-processed-transaction
  - get-transaction-report
  - get-latest-tx-using-account
  - get-arch-txid-from-btc-txid
- Network/State
  - get-current-state
  - get-network-pubkey
  - reset-network
  - start-dkg

> See method pages in the subfolders for details and request/response examples.
